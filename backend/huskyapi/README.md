Play Seed
=================================
This seed project based on Play 2.5 with Scala. Should help you easy to understand Play structure. 


Prerequsit
===========
- Intelli J 2016
- JDK 8
- sbt
- Please add conf "huskytrip.key" to localhost.conf (create file by yourself) 


Command
=====
- sbt compile
- sbt test
- sbt run
- sbt dist
